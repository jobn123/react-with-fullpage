# react-with-fullpage

a template of [fullpage.js](http://alvarotrigo.com/fullPage/) in react.

# Usage

```
1. npm i react-with-fullpage && cd react-with-fullpage
2. npm install
3. npm run  dev
4. then type localhost:8080 in your browser
```
